Description de la compétition

Dans le cadre de la transition écologique actuelle, Orange s'est fixé plusieurs objectifs :
  • Pour 2025 : réduire de 30% les émissions de CO2 et augmenter de 50% la production d'énergie d'origine renouvelable.
  • Pour 2040 : Devenir Net Zéro Carbone.
Dans ce contexte, il est dans l'intérêt d'Orange de rendre plus verte la production d'énergie au niveau des sites mobiles de toute la France.
Pour cela, il faut localiser les sites mobiles Orange sur toute la France et déterminer si le site est solarisable (=possibilité d'installer des panneaux solaires).
La compétition proposée vise dans un premier temps à détecter différents objets sur une image satellite et dans un second temps de décider si le site mobile est solarisable.

Description des données mises à disposition du candidat

Les données mises à disposition au candidat se composent de 1200 images satellites des sites mobiles Orange et des annotations correspondantes aux bounding box locallisants les antennes. Les images satellites sont au format PNG et les annotations au format CSV.
Les images labellisées ont été sélectionné sur toute la France, il y a donc une grande variété d'environnement et de forme d'antennes.

Les données sont téléchargeables à l'aide de ce lien : DONNEES

Phase 1 : Détection d'antennes sur des images satellites

Dans le cadre d'une première phase, le candidat aura pour objectif de détecter une antenne sur une image satellite. Sur une image, il peut y avoir plusieurs antennes. Le candidat est libre d'utiliser toute méthode qui semble lui convenir pour résoudre cette phase. Il lui est toutefois conseillé d'utiliser des méthodes de Transfert Learning.
Les modèles de détection d'objets utilise une bounding box pour localiser un objet. Dans le dataset qui est proposé au candidat, les bounding box sont locallisées par les coordonnées xmin, ymin, xmax et ymax. Voici un shéma illustrant ce concept :

Phase 2 : Détection des arbres et des batiments sur des images satellites

Dans le cadre d'une seconde phase, le candidat aura pour objectif de détecter les obstacles environnant le site mobile sur une image satellite. Dans ce contexte, il sera demandé au candidat de détecter les abres et les batiments qui sont les principaux obstacles dans l'environnement d'une antenne.
Le candidat est libre d'utiliser toute méthode qui semble lui convenir pour résoudre cette phase. Il lui est toutefois conseillé d'utiliser des méthodes de Transfert Learning.
